### Running

Rename `sample.env` to `.env.local` and fill in the details.

```
# install
yarn install

# run in development mode
yarn dev
```

### Running with docker

```
docker-compose up
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
